# TaskManager - Test Credentials

## Default User Accounts

The application comes with pre-seeded test accounts:

### Administrator Account
- **Email:** admin@taskmanager.com
- **Password:** Admin@12345!
- **Role:** Administrator
- **Access:** Full access to all features

### Manager Account
- **Email:** manager@taskmanager.com
- **Password:** Manager@123!
- **Role:** Manager
- **Access:** Can create and manage tasks and groups

### Regular User Account
- **Email:** user@taskmanager.com
- **Password:** User@123456!
- **Role:** User
- **Access:** Can view and work on assigned tasks

## Sample Data

The application includes sample data:
- **2 Groups:** Marketing, Development
- **3 Tasks:** With different statuses and priorities
- **2 Task Logs:** Sample activity logs

## Getting Started

1. Run the application
2. You will be redirected to the login page
3. Log in with one of the accounts above
4. Navigate through:
   - **Tasks** - View and manage tasks
   - **Groups** - View and manage groups
   - **Activity Logs** - View task history

## Features

- Role-based access control (Administrator, Manager, User)
- Task management with status tracking
- Group organization for teams
- Activity logging for audit trail
- User authentication with ASP.NET Core Identity
